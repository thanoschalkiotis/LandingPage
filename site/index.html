Μορφώστε τα παιδιά σας μέσα από τον προγραμματισμό


“ Όλα τα πράγματα είναι δύσκολα πρωτού γίνουν εύκολα.”

-- Τόμας Φούλερ - Άγγλος μελετητής, ιεροκήρυκας και συγγραφέας

Μία από τις μεγαλύτερες ευλογίες στην οποία μπορεί να ελπίζει κανείς είναι να βλέπει το παιδί του να ευημερεί. Σε μια απαιτητική εποχή όπως αυτή που διανύουμε, ένα άτομο που δεν γνωρίζει τον τρόπο να χρησιμοποιήσει υπολογιστές θεωρείται αναλφάβητο. Γιατί όχι λοιπόν να μην κάνουμε ένα βήμα παραπέρα και να διδάξουμετα παιδιά μας πώς να προγραμματίζουν τα ίδια υπολογιστές.
